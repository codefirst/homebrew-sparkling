Homebrew Sparkling

Install iOS apps without AppStore/Jailbreak.

Install now

INSTALL

  1. Install Xcode 7.
  2. Launch Xcode, and go to "Preferences" -> "Accounts", register your AppleID.
  3. Click "View Details" at right bottom, and create iOS Development signing Identifies.
  4. Install homebrew
  5. Paste following at a Terminal prompt.
brew install codefirst/sparkling/brew-sparkling --HEAD

FEATURES

Open Source

You can install "free" iOS apps. And more, you can easily add, and modify apps.

No Crack

You can install apps without cracked iPhone. So you can still use offical iOS apps.

No Developer Program

You need not to join Apple Developer Program.

USAGE

Install Firefox for iOS

Launch Xcode and type following:

brew sparkling install firefox

Search Firefox

brew sparkling search firefox

Check your environment

# show all accounts
brew sparkling accounts

# show all cerificates
brew sparkling certificates

# show all connected devices
brew sparkling devices

Configure your environment

If you have multi AppleID or certificates, you should configure which you use at ~/.profile

# set your account
export BREW_SPARKLING_USERNAME=brew_sparkling@icloud.com

# set your certificate
export BREW_SPARKLING_CERTIFICATE='iPhone Developer: brew_sparkling@icloud.com (xxxxxx)'

# set your device
export BREW_SPARKLING_DEVICE=mzpPhone

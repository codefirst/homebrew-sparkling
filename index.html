Homebrew Sparkling

Install iOS apps over Lightning without AppStore/Jailbreak.

Install now

INSTALL

brew install codefirst/sparkling/brew-sparkling --HEAD

Install homebrew, and paste that at a Terminal prompt.

FEATURES

Open Source

You can install "free" iOS apps. And more, you can easily add, and modify apps.

No Crack

You can install apps without cracked iPhone. So you can still use offical iOS apps, too.

No Developer Program

Without joinning Apple Developer Program, you can install iOS apps.

USAGE

Install SparklingSample:

brew sparkling install SparklingSample

Search SparklingSample:

brew sparkling search SparklingSample
